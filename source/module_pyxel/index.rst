Jeux avec interface graphique
=============================

La création d'un jeu avec une interface graphique est souvent un objectif pour les développeurs. C'est souvent se lancer dans un code long, difficile et dont il faut gérer les bugs. Il existe en Python de nombreux modules qui permettent d'utiliser une interface graphique comme ``Tkinter``, ``Pygame``, ``Qt`` ou ``Arcade``. Dans les projets de jeux que nous allons développer, on utilisera le module ``Pyxel``.

Ce module permet de créer des jeux rétros. Son principal avantage est sa simplicité d'utilisation. Pour démarrer, il me semble adapté aux jeux développés et sa prise en main est aisée. De plus, vous trouverez de la documentation et des tutoriels sur le site du jeu concours **La nuit du code``.
